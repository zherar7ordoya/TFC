CapturadorLogic Métodos

Métodos

CotizarServicio Cotiza un servicio de mudanza en base a la fecha, categoría y cantidad de kilómetros.
CrearTransaccion Crea una transacción a partir de un cliente y una orden.
DistanciarDesdeCentral Calcula la distancia entre la central y una ubicación en el mapa usando el algoritmo de Manhattan.
DistanciarEntreLocaciones Calcula la distancia entre dos ubicaciones en el mapa usando el algoritmo de Manhattan.
ImprimirOrden Crea un documento imprimible a partir de un cliente y una orden.
ObtenerCamionesDisponibles Obtiene una lista de camiones disponibles para una fecha dada.
ObtenerClientesActivos Obtiene una lista de clientes activos (no bloqueados o eliminados).
ObtenerLocacionesActivas Obtiene una lista de locaciones activas (no bloqueadas o eliminadas).
ValidarOrden Valida una entidad y muestra un mensaje de error si no es válida.
